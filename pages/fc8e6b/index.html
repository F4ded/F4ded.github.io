    Github+PicGo搭建专属于你的图床

    # 搭建

    # Github

    1. 在github上新建一个仓库，注意把类型设置成"public"。

    1. 生成个人token：右上角点击头像->"settings"->"Developer settings"->"Personal access tokens"，原来没有token新建一个就可以。

    # Picgo

    • 下载地址：https://github.com/Molunerfinn/PicGo/releases

    点击Github图床设置

    • 设定仓库名：Github用户名/仓库名称
    • 设定分支名：master
    • 设定Token：复制之前生成的Token，粘贴过来就可以
    • 指定储存路径：图片在仓库里的存储路径，可以自己指定
    • 设置自定义域名：https://cdn.jsdelivr.net/gh/[Github]用户名/仓库名

    # Typora设置

    文件->偏好设置，再按照如下设置即可

    设置完成后点击“验证图片上传选项”即可，如果出现类似下面提示，则可以正常使用。

    # 可能的错误

    如果上传图片显示{“success”,false}，可能是图片名称冲突的原因。

    解决方法：打开PicGo设置，将”时间戳重命名“打开即可。
